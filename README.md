# 📘 小白也能用的 Clash 链式代理配置 (Chain Proxy)

如果你购买了 **静态住宅 IP** (用于 ChatGPT/TikTok)，但直连速度很慢或者连不上，这个配置就是为你准备的。
它能让你买的 **“好机场”** 带着你的 **“静态 IP”** 一起飞，既快又安全。

## 🚀 快速开始

### 第一步：准备工作
1.  下载软件：**Clash Verge** (电脑) 或 **Clash Meta for Android** (手机)。
2.  准备好两个东西：
    * 你的 **机场订阅链接** (平时用来翻墙的那个)。
    * 你的 **静态 IP 信息** (商家发给你的 IP、端口、账号、密码)。

### 第二步：如何获取填空题的答案？(关键步骤！)
很多小白卡在不知道 `server`, `cipher` 这些东西去哪找。其实很简单：

1.  **复制你的机场订阅链接**。
2.  在浏览器地址栏粘贴并回车，你会下载一个文件（通常是一串乱码名字），或者看到满屏文字。
3.  **用记事本 (Notepad) 打开这个文件**。
4.  搜索 `proxies:`，你会看到类似下面的一段代码：
    ```yaml
    - {name: "香港节点1", server: "hk.line1.com", port: 1234, type: ss, cipher: aes-128-gcm, password: "xxxx"}
    ```
5.  **这就是答案！** 对照着把里面的内容，填入本项目 `config.yaml` 对应的位置。

### 第三步：修改配置文件
1.  下载本项目的 `config.yaml` 文件。
2.  用记事本打开它。
3.  找到 `proxies:` 区域，把上面找到的信息填进去：
    * `✈️ 入口-香港` -> 填入你机场里最好用的那个节点信息。
    * `🚀 链式-静态落地出口` -> 填入你购买的静态 IP 信息。
4.  保存文件。

### 第四步：导入使用
把改好的 `config.yaml` 拖进 Clash 软件，右键选择“启用”。
现在，去连接 ChatGPT 吧！你的 IP 已经变成了尊贵的原生住宅 IP，而且速度飞快！

## ❓ 常见问题 (Q&A)

**Q: 这里的 DNS 设置我要改吗？**
A: **千万别改！** 这里的 DNS 是指“域名解析服务器”，模版里已经填好了腾讯(119.29.29.29)和阿里(223.5.5.5)的公共服务器。它不是让你填网址的。
（PS：这里是AI帮弄的问答，其实并不全面，如果你的机场是用别的DNS，你也可以改，但最简单的方法是把模版和你的机场yaml文件丢给AI就行了。
你只需要告诉AI入口要哪几个节点就好了，这就要你自己事先测速，选最快最稳的几个节点作为链式代理的入口。

**Q: 为什么我填好了还是连不上？**
A: 请检查 `dialer-proxy: "🚪 链式-入口选择"` 这一行前面的缩进（空格）是否正确。YAML 格式对空格非常敏感，必须对齐。
